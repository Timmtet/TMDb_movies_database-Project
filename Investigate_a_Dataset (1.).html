Investigate a Dataset - Database_TMDb_movie_data Introduction

Dataset Description This data set contains information over 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. The database has 10866 rows and 21 columns. The columns include: Id, imdb_id, popularity, budget, revenue, original_title, cast, homepage, director, tangline, overview, runtime, genre, production_company, release_date, vote_count, vote_average, release_year, budject_adj, and revenue_adj.

Questions for analysis

  1. What is the association between movies with tagline and revenue/adjusted revenue(renvenue_adj)
  2. What decade are the most popular movies released; how much revenue was recorded that year?
